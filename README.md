# Coding World - Full Stack Portfolio

A modern full-stack portfolio website for Coding World with internship application system, AI chatbot, and responsive design.

## Features

- 🎨 Modern, responsive UI with dark/light mode
- 🤖 AI-powered chatbot with voice support
- 📝 Internship application system with file upload
- 📧 Automatic email notifications
- 💾 MongoDB Atlas database integration
- 🗺️ React Router for navigation
- 🔒 Secure file upload with Multer
- 📱 Mobile-friendly design

## Tech Stack

- **Frontend**: React.js, React Router, CSS-in-JS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB Atlas
- **File Upload**: Multer
- **Email**: Nodemailer
- **Styling**: Custom CSS with animations
